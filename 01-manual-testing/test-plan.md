# QA Test Plan – SauceDemo E-Commerce Application

## 1. Introduction
This test plan defines the scope, approach, and testing activities for validating the SauceDemo e-commerce application.

## 2. Objectives
- Validate core user flows (login, product browsing, cart, checkout)
- Identify functional and UI defects
- Ensure system behaves correctly under valid and invalid inputs

## 3. Scope

### In Scope:
- User Login
- Product Listing Page
- Add/Remove Items from Cart
- Checkout Process

### Out of Scope:
- Performance testing
- Security testing (advanced)

## 4. Test Approach
- Manual Functional Testing
- Positive and Negative Testing
- Exploratory Testing

## 5. Test Environment
- Browser: Google Chrome (latest)
- OS: Windows 10
- Application: https://www.saucedemo.com

## 6. Entry Criteria
- Application is accessible
- Test data available (valid/invalid users)

## 7. Exit Criteria
- All critical test cases executed
- No open high severity bugs
